# 課題内容
自分のポートフォリオサイト（自己紹介サイト）を作る
# 要件
以下の情報を含んでいること
- 自分の名前
- 自分の連絡先
- 自分のgithubアカウントへのリンク
    - 例 : https://github.com/example
- 自分のスキル
    - 何もない、と思う人は以下を記載すること
        - novelsphere.js
        - HTML
        - CSS
        - JavaScript
- 自分の活動履歴
    - 自分が関わったプロジェクトを可能な限り記述する
        - 多いほうが良いですが、網羅的に記述する必要はありません
        - それぞれ最低でも、以下の中身を持つ
            - プロジェクト名
            - 概要
            - 使用したスキル
            - 自分の役割
        - 何もない、と思う人は以下を記載すること
            - プロジェクト：本ポートフォリオサイト
            - 概要：授業の中で、このサイトを制作しました。
            - 使用したスキル：HTML,CSS
            - 自分の役割：一人で制作を完結させた。
- なんらかのCSSを適応していること
# 評価ポイント
- 以下を含まれていると高評価
    - きちんと構造化されたHTML/CSS/JavaScriptであること
    - 無意味な記述がないこと（適応されないCSSが存在するなど）
    - インデントがきちんとなされ読みやすくなっていること
        - 極度に厳格なルールである必要はありません
    - 拡張性が考慮されていること
    - 凝った内容・デザインであること
    - ただし、ページ数は評価に影響しない
- JavaScriptを使って、何らかのギミックが含まれていると高評価
    - jQueryを十分に扱えていないため、含まれていない場合でもA評価はあり得る
    - 思いつかない場合、以下のような例が考えられる
        - 「トップへ戻る」ボタンの設置
        - モーダルの設置
        - カルーセルの設置
        - ページ内タブ表示の実装
        - 画面サイズを取得し、画面中央への要素の出力
